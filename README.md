# Repositório de Estudos em JavaScript e Lógica

Este repositório foi desenvolvido para centralizar todos os meus exercícios, códigos práticos e algoritmos criados durante meus estudos de JavaScript, aplicados à área de Tecnologia da Informação.

O objetivo principal é demonstrar minha evolução em lógica de programação, manipulação de dados, estruturas de repetição e interatividade dinâmica no navegador.

---

### Tecnologias e Conceitos Praticados
* Lógica de Programação Estruturada
* Manipulação e Validação de Formulários (DOM)
* Estruturas de Repetição (for, while)
* Eventos em JavaScript (click, submit)

---

### Projetos Desenvolvidos

#### Projeto Formulário Aluno
Sistema de captura e validação de dados cadastrais de estudantes. Excelente para praticar a manipulação de elementos HTML via JavaScript e simular o recebimento de dados em sistemas escolares.
* Acesse o código: [Ver arquivos do projeto](./PROJETO-FORMULÁRIO-ALUNO)
* Acesse o projeto rodando: https://graziellemouradesouza.github.io/javascript-exercicios/PROJETO-FORMULÁRIO-ALUNO/

#### Calculadora de IMC
Aplicação interativa que calcula o Índice de Massa Corporal (IMC) a partir do peso e altura informados, exibindo a classificação da faixa de peso do usuário em tempo real utilizando estruturas condicionais.
* Acesse o código: [Ver arquivos do projeto](./calculadora-imc)
* Acesse o projeto rodando: https://graziellemouradesouza.github.io/javascript-exercicios/calculadora-imc/

#### Tabuada Dinâmica
Gerador automático de tabuada onde o usuário insere um número e o script calcula e renderiza instantaneamente os resultados de 1 a 10 na tela, aplicando conceitos fundamentais de laços de repetição.
* Acesse o código: [Ver arquivos do projeto](./tabuada)
* Acesse o projeto rodando: https://graziellemouradesouza.github.io/javascript-exercicios/tabuada/
